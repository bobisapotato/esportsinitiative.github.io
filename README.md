# What is esportsinitiative.github.io?
The 5th iteration of ESI website. Main goals:

- [x] Static site hosted on GitHub Pages
- [x] Easy content creation and editing
- [x] Pages for:
  - [ ] Events
  - [ ] Sponsors
  - [ ] Teams
  - [ ] Games
  - [ ] Short Links

All easily solved and maintained by ***Jekyll!***
